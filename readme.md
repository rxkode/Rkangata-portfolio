 Personal Portfolio Website 

A modern, fully responsive portfolio website built with HTML5, CSS3, and JavaScript to showcase professional skills, projects, and experience.

 Key Features

Responsive across all devices

Dark/Light mode with saved preference

Smooth scroll animations and transitions

Project filtering by category

Testimonial carousel

Validated contact form

Clean, modern UI/UX design

 Technologies

HTML5, CSS3 (Flexbox, Grid, Animations), JavaScript (ES6+), Font Awesome, Google Fonts

 Structure
index.html      → Main page
styles.css      → Styling
script.js       → Interactivity
images/         → Assets
README.md       → Documentation

 Setup

Clone repo → open index.html or use VS Code Live Server

Live demo available via GitHub Pages

 Customization

Edit CSS variables for theme colors

Replace placeholder text, images, and social links

Update About, Services, and Skills sections

 Website Sections

Hero (animated intro)

About (bio + skills)

Services (what you offer)

Portfolio (project gallery)

Testimonials (reviews)

Contact (form + info)

Footer (links + credits)

 Compatibility

Works on all major browsers including Chrome, Firefox, Safari, Edge, and mobile.
